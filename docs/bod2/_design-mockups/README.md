## 切替えたくなったら

1. このフォルダの該当モックを見直す
2. announcement.html / index.html / slide_part2.html / members.html / setup.html のカラートークン（`:root` の CSS 変数）を入れ替え
3. 装飾要素（提灯・波紋など）を該当案のモチーフに置換

採用デザイン（A）の CSS トークンは各ページの `:root` ブロックに定義済み。
